open(2) int open(const char *pathname, int flags); On success, open() returns the new file descriptor, or -1 if an error occurred (in which case, errno is set appropriately).
read(2) ssize_t read(int fd, void *buf, size_t count); On success, the number of bytes read is returned (zero indicates end of file). On error, -1 is returned, and errno is set appropriately.
write(2) ssize_t write(int fd, const void *buf, size_t count); On success, the number of bytes written is returned (zero indicates nothing was written). On error, -1 is returned, and errno is set appropriately.
close(2) int close(int fd); On success, close() returns zero. On error, -1 is returned, and errno is set appropriately.
ioctl(2) int ioctl(int fd, unsigned long request, ...); Usually, on success zero is returned. A few ioctl() requests use the return value as an output parameter and return a nonnegative value on success. On error, -1 is returned, and errno is set appropriately.
